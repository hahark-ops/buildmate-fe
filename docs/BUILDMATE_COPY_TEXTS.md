## 10. DM / 채팅 (`dm.html`, `js/dm.js`)
- title: 협업 채팅 - 빌드메이트
- 헤더 DM 버튼: 협업 채팅
- 패널 제목 `대화 목록` -> 협업 채팅
- 상태 문구 `실시간 1:1 대화를 지원합니다.` -> 프로젝트 논의를 위한 실시간 협업 채팅을 지원합니다.
- empty state `아직 시작한 대화가 없습니다.` -> 아직 시작한 협업 대화가 없습니다.
- 입력 placeholder `메시지를 입력하세요` -> 협업 메시지를 입력하세요.
- push 버튼 `알림 켜기` 유지
- push 힌트 `채팅방을 보고 있지 않을 때 브라우저 알림을 받습니다.` -> 프로젝트 관련 새 메시지를 브라우저 알림으로 받을 수 있습니다.
- 연결 중 상태: 실시간 연결 중...
- 연결 완료 상태: 실시간 협업 채팅이 연결되었습니다.
- 연결 끊김 상태: 실시간 연결이 끊어졌습니다.
- 재연결 상태: 실시간 연결을 다시 시도하고 있습니다.
- 방 empty state: 아직 참여 중인 협업 채팅방이 없습니다.
- 방 미선택 상태: 협업할 메이트를 선택해주세요.
- 최근 활동 없음 상태: 아직 협업 메시지가 없습니다.
- 방 preview fallback: 아직 협업 메시지가 없습니다.

## 12. README / 영상용 핵심 문장
- 빌드메이트는 사이드 프로젝트 팀원을 찾고, 공개 질문과 실시간 협업 채팅으로 협업을 시작할 수 있는 팀 빌딩 서비스입니다.
- 초기 타깃은 AI 툴을 활용해 빠르게 MVP를 만들고 싶은 메이커이지만, 서비스의 핵심은 팀원 모집과 협업 시작에 있습니다.
- 이 서비스의 핵심은 모집글 작성 -> 공개 반응 수집 -> 실시간 협업 채팅으로 이어지는 흐름입니다.
