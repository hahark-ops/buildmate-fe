# Buildmate Copy List

## 서비스명
- 아무 말 대잔치 -> 빌드메이트

## 핵심 카피
- 혼자 만들다 막히면, 함께 만들 메이트를 찾으세요.
- AI 툴로 만드는 사이드 프로젝트, 이제 메이트와 함께 시작하세요.
- 팀원을 찾고, 대화를 시작하고, 프로젝트를 함께 만들어보세요.

## 메인 페이지
- 제목: 빌드메이트
- 서브카피: AI 툴을 활용해 사이드 프로젝트를 만드는 메이커들이 팀원을 찾고 실시간으로 협업을 시작할 수 있는 커뮤니티
- 글쓰기 버튼: 프로젝트 모집하기
- 빈 상태: 아직 등록된 프로젝트가 없습니다. 첫 모집글을 올려보세요.

## 로그인 페이지
- 제목: 다시 빌드를 시작해볼까요?
- 설명: 메이트를 찾고, 협업 대화를 시작하고, 프로젝트를 함께 만들어보세요.

## 회원가입 페이지
- 제목: 빌드메이트 시작하기
- 설명: AI 툴과 함께 만드는 당신의 프로젝트, 이제 메이트와 함께 진행해보세요.
- 성공 문구: 회원가입이 완료되었습니다. 로그인 화면으로 이동합니다.

## 게시글 관련
- 게시글 -> 프로젝트 모집글
- 게시글 작성 -> 프로젝트 모집하기
- 제목 placeholder: 예: AI 가계부 앱 같이 만들 팀원 구합니다
- 본문 안내: 프로젝트 소개, 현재 진행 상태, 필요한 역할, 원하는 협업 방식을 적어주세요.

## 댓글 관련
- 댓글 placeholder: 질문이나 관심 표현을 남겨보세요.
- 댓글 등록 버튼: 질문/댓글 등록 또는 그대로 유지

## 좋아요 관련
- 좋아요 -> 관심
- 좋아요수 -> 관심 수

## DM 관련
- DM -> 협업 채팅
- 실시간 1:1 대화를 지원합니다. -> 실시간 협업 채팅을 지원합니다.
- 아직 시작한 대화가 없습니다. -> 아직 시작한 협업 대화가 없습니다.
- 메시지를 입력하세요 -> 협업 메시지를 입력하세요

## 프로필 관련
- 내 정보 -> 메이커 프로필
- 프로필 수정 -> 프로필 / 협업 정보 수정

## 탈퇴 관련
- 영구 삭제 문구는 유지하되, 프로젝트/댓글/DM 기록이 함께 삭제된다는 점을 명확히 유지
