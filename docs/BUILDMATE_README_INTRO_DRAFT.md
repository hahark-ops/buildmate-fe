# Buildmate README Intro Draft

## 프로젝트 소개
빌드메이트는 AI 툴을 활용해 사이드 프로젝트를 만드는 메이커들이 팀원을 찾고, 실시간으로 협업을 시작할 수 있는 커뮤니티 서비스입니다.

## 왜 만들었는가
사이드 프로젝트를 혼자 시작하는 사람은 많지만, 실제로 끝까지 만드는 과정에서는 역할 분담과 대화가 필요합니다. 특히 바이브 코딩 입문자나 AI 툴을 활용하는 메이커는 빠르게 시도하고 검증할 수 있지만, 혼자 진행하다 막히는 경우가 많습니다. 빌드메이트는 이런 사용자가 프로젝트 모집글을 올리고, 관심 있는 메이커와 댓글과 DM으로 바로 연결될 수 있도록 만들었습니다.

## 핵심 기능
- 프로젝트 모집글 작성 및 조회
- 댓글과 관심 표시를 통한 초기 반응 수집
- 실시간 1:1 DM과 unread/read 처리
- Redis pub/sub 기반 분산 DM 구조
- Web Push 기반 부재 알림 구조
- Docker, EC2, Lambda, k3s, rolling update, blue-green, 멀티 클러스터 증빙
- pytest + Playwright + CI 게이트 기반 품질 관리

## 한 줄 설명
