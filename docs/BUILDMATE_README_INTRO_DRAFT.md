# Buildmate README Intro Draft

## 프로젝트 소개
빌드메이트는 사이드 프로젝트를 함께 만들 팀원을 찾고, 공개 질문과 실시간 협업 채팅으로 협업을 시작할 수 있는 팀 빌딩 서비스입니다.

초기 타깃은 AI 툴을 활용해 빠르게 MVP를 시도하는 메이커이지만, 서비스의 본질은 `팀원 모집과 협업 시작`에 있습니다.

## 왜 만들었는가
사이드 프로젝트를 시작하는 사람은 많지만, 실제로 끝까지 만들기 위해서는 역할 분담과 빠른 커뮤니케이션이 필요합니다. 특히 바이브 코딩 입문자나 AI 툴을 활용하는 메이커는 빠르게 시도하고 검증할 수 있지만, 혼자 진행하다가 막히는 경우가 많습니다.

빌드메이트는 이런 사용자가 프로젝트 모집글을 올리고, 관심 있는 메이커와 공개 질문을 주고받고, 실시간 채팅으로 바로 협업 논의를 시작할 수 있도록 만들었습니다.

## 제품 관점에서의 핵심 가치
- 프로젝트를 함께 만들 메이트를 찾을 수 있다.
- 공개 질문과 관심 표현을 통해 초기 반응을 받을 수 있다.
- 실시간 채팅으로 구체적인 협업 논의를 바로 시작할 수 있다.
- 모집글, 질문, 관심, DM이 하나의 팀 빌딩 흐름으로 이어진다.

## 기술적 강점
- 실시간 1:1 DM과 unread/read 처리
- Redis pub/sub 기반 분산 DM 구조
- Web Push 기반 부재 알림 구조
- Docker, EC2, Lambda, k3s, rolling update, blue-green, 멀티 클러스터 증빙
- pytest + Playwright + CI 게이트 기반 품질 관리
