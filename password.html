Account Security

계정 보안 흐름을 최신 상태로 유지하세요.

현재 비밀번호 확인, 새 비밀번호 정책, 변경 이후 체크 포인트를 같은 화면에서 안내합니다. 운영 중인 계정일수록 주기적인 정리가 중요합니다.

로그인 이메일 이메일을 불러오는 중입니다.
8-20자 대/소문자 포함 숫자 포함 특수문자 포함
현재 비밀번호 확인 강한 정책 유지 변경 후 즉시 반영
현재 비밀번호 확인 기존 비밀번호를 먼저 확인해 실수로 인한 계정 잠금과 오입력을 줄입니다.
강한 비밀번호 정책 대문자, 소문자, 숫자, 특수문자를 모두 포함하는 비밀번호만 허용됩니다.
